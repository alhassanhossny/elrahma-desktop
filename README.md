# Elrahma HR Desktop

A simple employee management desktop application built with C# for Windows 7 & 10.

## Prerequisites

- .NET 6 SDK or later
- Visual Studio 2022 or VS Code with C# extensions

## Setup Instructions

1. Install .NET 6 SDK from https://dotnet.microsoft.com/download/dotnet/6.0
2. Clone the repository
3. Restore packages: `dotnet restore`
4. Build the project: `dotnet build`
5. Run the application: `dotnet run`

## Target Framework

This application targets .NET 6.0 for maximum compatibility with Windows 7 and 10.
